# Compressor — Portable Advanced Video Encoding for Final Cut Pro

## About Compressor

Compressor provides a complete video encoding environment for Final Cut Pro editors and video professionals. Compressor combines batch export, custom settings, distributed rendering, and advanced codecs. Compressor enables users to encode video for Apple devices, web streaming, broadcast, and DVD/Blu‑ray. Compressor runs on desktop platforms. Compressor is trusted by video editors, post‑production houses, YouTubers, broadcast engineers, and Final Cut Pro users worldwide.

## Deep Dive — Video Encoding Notes

Formats: Apple devices, web (H.264, HEVC), ProRes, MXF, MPEG‑2, MPEG‑4
Batch export: add multiple jobs, apply settings, encode in one batch
Distributed rendering: use multiple Macs to encode faster (Compressor cluster)
Frame controls: resize, deinterlace, retiming, pulldown, clean aperture
Image sequences: TIFF, PNG, OpenEXR, DPX, TGA, JPEG image sequences
